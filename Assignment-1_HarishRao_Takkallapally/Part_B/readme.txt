Tags Used:
<title>: The title of the document, displayed in the browser's title bar or on the tab
<link rel="icon" href="favicon.ico" type="image/x-icon">: Links to a small icon that is displayed in the browser's tab
<a>: Defines a hyperlink, which allows users to navigate to a different webpage or a specific location on the same webpage
<h1>: Defines a heading
<img>: Displays an image on the webpage
<table>: Creates a table
<tr>: Creates a row in the table
<th>: Creates a header cell in the table
<td>: Creates a standard cell in the table
<footer>: Represents a container for the footer of a document or section
<form>: Used to create an HTML form for user input.
<button>: This defines a clickable button.
